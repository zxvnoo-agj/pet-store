---

## 一、品牌概述

皇家猫粮（Royal Canin）由法国兽医师 Jean Cathary 于1968年创立，是全球领先的健康营养宠物食品品牌。品牌秉持"精准营养"理念，根据不同**年龄、体型、品种及生活方式**，开发了超过750种专属配方。1995年进入中国市场，在中国提供300多种营养组合产品，服务超过10000家宠物医院。

### 核心 ingredient 理念

- **功能性优先**：选择原料基于其营养成分和对成品饮食的作用，而非仅凭听起来是否"天然"
- **高消化性蛋白质（L.I.P.）**：严选消化率90%以上的蛋白质
- **精准营养配方**：通过营养科学和观察研发，而非追逐市场趋势

## 六、处方粮系列（Veterinary Diet）

> **注意**：处方粮需在兽医指导下使用

### 6.1 泌尿道处方粮（Urinary S/O）

**适用症状**：
- 鸟粪石（磷酸铵镁结石）的溶解与预防
- 草酸钙结石预防
- 猫下泌尿道疾病（FLUTD）

**核心机制**：
- 尿液酸化
- 低镁配方
- 降低相对饱和度（RSS），减少结晶形成
- 稀释尿液，促进定期膀胱冲洗

**产品系列**：
- **Urinary S/O**：标准版成猫泌尿道处方干粮
- **Urinary S/O Moderate Calorie**：适量卡路里版，适合绝育后或超重猫

## 十、选购建议

### 10.1 按猫咪情况选择

| 猫咪情况 | 推荐产品 |
|---------|---------|
| 离乳期幼猫（1-4月） | Mother & Babycat 慕斯/奶糕干粮 |
| 成长期幼猫（4-12月） | Kitten 幼猫干粮 + 幼猫湿粮 |
| 健康成猫（1-7岁） | Fit 32 / 或根据生活方式选择 |
| 室内猫 | Indoor 27 系列 |
| 消化敏感猫 | Sensible 33 或 Digestive Care |
| 7岁以上熟龄猫 | Indoor 7+ / Ageing 12+ |
| 长毛猫/易吐毛球 | Hairball Care 或含洋车前子配方 |
| 绝育后易发胖 | Weight Care 或绝育专用配方 |
| 品种猫 | 对应品种专用粮 |
| 泌尿问题（需兽医确认） | Urinary S/O 处方粮 |
| 肾脏问题（需兽医确认） | Renal / Early Renal 处方粮 |
| 肠胃问题（需兽医确认） | Gastrointestinal 处方粮 |

### 10.2 干湿粮搭配建议

- **幼猫期**：建议干粮+湿粮混合喂养，湿粮补充水分
- **成猫期**：可以干粮为主，搭配湿粮增加水分摄取
- **老年猫**：建议增加湿粮比例，帮助水分摄取和肾脏健康
- **泌尿问题猫**：务必增加湿粮比例，稀释尿液

---

## 十一、注意事项

1. **换粮需循序渐进**：建议7-10天内逐步过渡，避免肠胃不适
2. **处方粮需遵医嘱**：泌尿道、肾脏、肠胃处方粮需在兽医指导下使用
3. **确保饮水充足**：尤其是以干粮为主食的猫咪
4. **注意保质期**：购买时检查生产日期和保质期
5. **存储条件**：存放于阴凉干燥处，开封后尽快使用
6. **定期体检**：建议每年进行健康检查，根据猫咪健康状况调整饮食
