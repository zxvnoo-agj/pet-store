## 一、品牌概述

### 1.1 品牌背景

鲜朗（ROSY FRESH）是浙江吉宠商贸有限公司旗下的高端宠物食品品牌，自2017年开始投入研发低温烘焙宠食工艺，历经三年技术打磨，于2020年正式将产品推向市场。2023年，鲜朗在河南漯河搭建自有工厂（鲜朗宠物食品漯河有限公司），实现从研发到生产的全链路品控。2024年，鲜朗连续3年获得犬猫烘焙主粮TOP双第一，成为天猫平台年度销冠品牌。

### 1.2 核心产品理念

- **"0肉粉+低温烘焙"**：坚持不使用肉粉，全部采用鲜肉或冷冻保鲜肉直接入料
- **全鲜肉配方**：原料以新鲜鸡肉、鸭肉、兔肉、深海鱼等为主
- **极简纯净配方**：不添加人工诱食剂、着色剂、化学防腐剂
- **原料透明**：公开主要原料供应商信息，肉源可追溯

### 1.3 核心工艺——低温烘焙

| 对比维度 | 低温烘焙工艺（鲜朗） | 传统高温膨化工艺 |
|---------|-------------------|----------------|
| 加工温度 | 60℃~90℃ | 120℃~180℃ |
| 营养保留 | 更好保留维生素、Omega-3等热敏性营养 | 高温导致部分营养流失 |
| 颗粒质地 | 酥脆干爽，表面不喷油 | 表面可能喷涂油脂 |
| 淀粉需求 | 较低，无需大量淀粉膨化成型 | 需要较高淀粉含量辅助膨化 |
| 消化率 | 较高，颗粒酥脆易消化 | 相对较低 |

## 三、烘焙粮系列（核心产品线）

鲜朗烘焙粮是其最核心的产品，全部采用**90℃低温烘焙工艺**，0肉粉添加，无谷配方。以下逐一详解各配方。

### 3.1 禽肉配方烘焙粮（经典入门款）

**定位**：入门级首选，温和低敏，适合大多数猫咪

**适用对象**：全阶段猫咪（成猫、幼猫）

**核心特点**：
- 鲜鸡肉+鲜鸭肉双源组合，低敏易消化
- 适合新手铲屎官或希望猫咪结实长肉的家庭
- 颗粒为中等圆形，干爽不油腻，预防黑下巴

**主要成分**：
- 冷冻保鲜鸡肉（73%~75.5%）
- 冷冻保鲜鸭肉（11%）
- 鲜鸡肝（5%~8%）
- 红薯颗粒（帮助成型）
- 纤维素（木质纤维+苜蓿草）
- 鱼油、全脂羊奶粉、蛋黄粉
- 蓝莓干、覆盆子干、车前子、菊苣根粉

**营养分析保证值**：

| 营养成分 | 含量 |
|---------|------|
| 粗蛋白质 | ≥40%（实测约43%） |
| 粗脂肪 | ≥18%（实测约22%） |
| 粗纤维 | ≤4% |
| 粗灰分 | ≤10%（实测约8.2%） |
| 钙 | ≥1.0% |
| 总磷 | ≥0.8% |
| 牛磺酸 | ≥0.2% |
| 水分 | ≤10% |
| 淀粉 | 约4%（马铃薯粉） |

**实测数据（第三方检测）**：
- 粗蛋白实测43.09%，粗脂肪实测22.0%
- 钙磷比1.5:1，合理
- 未检出黄曲霉毒素、沙门氏菌、BHA/BHT等有害物质

**包装规格**：1kg、2kg、4kg、6kg、8kg、12.8kg

**颗粒形状**：圆形，直径约1×0.9cm，适配自动喂食器

## 十、品牌优势与注意事项

### 10.1 核心优势

1. **自有工厂品控**：2023年搭建自有工厂，非代工模式，品控更稳定
2. **原料透明可追溯**：肉源来自双汇、益客等大厂，供应商信息公开
3. **0肉粉0谷物0豆类**：配方纯净，减少过敏源
4. **第三方检测**：检测报告公开，未检出有害物质
5. **颗粒干爽**：烘焙工艺不喷油，有效预防黑下巴
6. **适口性良好**：多数猫咪接受度高，换粮顺利

### 10.2 注意事项

1. **保质期较短**：因鲜肉含量高，保质期通常为12个月，开封后尽快食用
2. **价格中高端**：均价约35-40元/斤，比膨化粮贵
3. **换粮需过渡**：建议7-10天循序渐进，避免肠胃不适
4. **适口性个体差异**：部分猫咪对天然风味需要时间适应
5. **存储条件**：存放于阴凉干燥处，避免受潮
